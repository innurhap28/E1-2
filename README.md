## 퀴즈 주제 선정 이유

주제 : 나(김시현)에 대한 퀴즈
선정 이유 : 피평가자인 나 자신을 주제로 퀴즈를 구성하여, 동료들이 평가를 함에 있어 흥미를 가지고 참여하는 것이 가능하다고 생각했으며, 편안한 분위기 속에서 보다 솔직하고 적극적인 피드백이 이루어질 수 있을 것이라고 판단했습니다.
